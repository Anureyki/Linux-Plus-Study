# Linux+ (XK0-006) Study Repository

## Exam Overview
- **Exam version:** V8 (XK0-006)
- **Launch date:** July 15, 2025
- **Questions:** Maximum of 90 (multiple-choice + performance-based)
- **Duration:** 90 minutes
- **Passing score:** 720 (scale of 100–900)

## Skills Learned (What the exam validates)
- Configure and manage Linux systems, storage, networks, and services in cloud and hybrid environments
- Apply security best practices (permissions, authentication, firewalls, system hardening)
- Automate administration tasks with shell scripting, Python, and configuration management tools
- Deploy, maintain, and monitor containers and virtual machines
- Troubleshoot system, network, security, and application issues

## Exam Domains (What to study)

### System Management (23%)
- Boot process, kernel, filesystems, architectures
- Kernel modules, hardware components, device utilities
- LVM, RAID, partitions, mounted storage
- Hosts, DNS, interfaces, network tools
- Shell navigation, editing, redirection, environment variables
- Archiving, compression, data recovery
- Hypervisors, VMs, disk images
- Performance bottlenecks, resource optimization

### Security (18%)
- Permissions and authentication
- Firewalls and system hardening
- Encryption and access control
- Threat detection
- Secure system configuration

### Automation, Orchestration & Scripting (17%)
- Shell scripting
- Python automation
- Configuration management tools (Ansible, etc.)
- Container deployment and management
- Virtual machine orchestration

### Services & User Management (20%)
- User account management
- Service configuration (systemd)
- Process management
- Application deployment and maintenance

### Troubleshooting (22%)
- System issues
- Network connectivity problems
- Security incident response
- Application failures
- Log analysis and resolution

## Key Technologies to Know
- **Systemd** (services, targets, journalctl)
- **Containerization** (Docker, podman, Kubernetes basics)
- **Automation** (Bash scripting, Python, Ansible)
- **Security** (SELinux, AppArmor, UFW, SSH hardening)
- **Storage** (LVM, RAID, partitions, mount points)
- **Networking** (DNS, interfaces, routing, firewalls)
- **Troubleshooting** (logs, strace, performance monitoring)
